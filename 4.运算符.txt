+ : 加(如果是数字,则数字相加;如果是字符串,则字符串相连接)
- : 减(所有运算子一律转为数值，再进行相应的数学运算)
* : 乘(所有运算子一律转为数值，再进行相应的数学运算)
/ : 除(所有运算子一律转为数值，再进行相应的数学运算)
i++ : 自加1
i-- : 自加1
n % 2 : 求余
= : 赋值运算符(如果是对象,则是引用)
> : 比较值1是否大于值2
>= : 比较值1是否大于等于值2
< : 比较值1是否小于值2
<= : 比较值1是否小于等于值2
== : 比较两个值是否相等
=== : 比较两个值的值和类型是否全都相同
!= : 比较两个值是否不相等
!== : 比较两个值的值或类型是否不相同
! : 取反运算符(结果自动转换为布尔值)
&& : 且运算符(如果第一个运算子的布尔值为true，则返回第二个运算子的值（注意是值，不是布尔值）；如果第一个运算子的布尔值为false，则直接返回第一个运算子的值，且不再对第二个运算子求值。)
|| : 或运算符(如果第一个运算子的布尔值为true，则返回第一个运算子的值，且不再对第二个运算子求值；如果第一个运算子的布尔值为false，则返回第二个运算子的值)
?: : 三元运算符:(如果第一个表达式的布尔值为true，则返回第二个表达式的值，否则返回第三个表达式的值。)例子: a < b ? c = a : c = b;

运算顺序:
优先级从高到低依次为：小于等于（<=)、严格相等（===）、或（||）、三元（?:）、等号（=）。
